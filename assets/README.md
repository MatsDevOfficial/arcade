# Assets

## Banner (Hack Club Ship)

Ship requires a **banner image** uploaded separately (not taken from a devlog).

1. Open `banner.svg` in a browser or Inkscape/Figma.
2. Export as **PNG** at **1200×630** → save as `banner.png` in this folder.
3. Upload `banner.png` on the Ship project page.

You can also reference `banner.svg` in the README (GitHub renders SVG).
