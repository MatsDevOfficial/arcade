## About

This project started as a way to scramble old part i got from an ACTION arcade, so I wanted to make an arcade. But since I am an absolute noob at programming I decided to make a smart assistant. Afther that I thought it would be a great to add things about my plants. In the future I want to make the assistant "smart", but for now I want to ship and fix the basics.

## PCB and CAD choices

For this project I have decided that I do NOT want to use a pcb. That is bc I want to be able to add as much expantions as I want, which I just cant with a pcb.

CAD is a different thing. I very sure want some enclosures, but do not have the full specs yet, so will make the CAD when I get those dimensions.
